# Experiment artifact cleanup — 2026-07-17

## Outcome

- Checkout size: approximately 3.2 GiB before, 1.1 GiB after.
- Deleted: 134 inventoried files totalling 2,355,500,837 bytes (2.194 GiB),
  plus small Python/macOS caches.
- Formal experiment results, final models, final resumable states, generated sample
  sets, contact sheets, discriminator reviews, configs, manifests, logs, histories,
  metrics, and reports were preserved.

## Removed artifacts

- All model/optimizer checkpoints from pipeline-only experiments `000`, `010`,
  `011`, `012`, and `021`.
- Periodic `training_state_epoch_*.pt` recovery snapshots from completed formal
  experiments `001`, `004`, and `005` after validating their final recovery state.
- Non-milestone fixed-noise epoch previews from completed formal experiments.
- `.DS_Store`, `.cache`, and Python bytecode cache directories found during cleanup.

## Recovery-state validation

- `001`: the retained legacy latest state records 2020 completed steps; its schema
  predates the explicit `completed_epoch` field, while `metrics.json` records the
  completed 20 epochs.
- `004`: the retained latest state records epoch 100 and 10,100 completed steps.
- `005`: the retained latest state records epoch 500 and 50,000 completed steps.

## Interpretation note

The raw `metrics.json` files are immutable run records. Their `artifacts` sections
describe everything produced at run completion and may therefore name periodic or
diagnostic checkpoints pruned by this later retention pass. This cleanup ledger and
`EXPERIMENT_SUMMARY.md` describe the current retained artifact surface.
